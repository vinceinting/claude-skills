## Task Format Reference

```
- [ ] `ID` [Priority/Effort] Description - Type
```

**ID Format:** `PROJECT-STREAM##`
- Stream: F (Frontend), B (Backend), G (General)
- Example: `KYLO-F01`, `KYLO-B12`, `KYLO-G03`

**Priority:**
- P0 = Critical / Blocker
- P1 = Important / Should have
- P2 = Nice to have / Can defer

**Effort:**
- Low = < 1 hour
- Medium = 1-4 hours
- High = > 4 hours

**Type:**
- Feature = New functionality
- Update = Enhancement to existing
- Bug = Defect fix
- Chore = Maintenance / docs / cleanup
